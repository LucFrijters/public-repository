# Example Configurations

This directory contains template configurations that can be customized for specific needs.

## Available Templates

- `docker-compose.template.yml` - Docker Compose template
- `package.template.json` - Node.js package.json template
- `nginx.template.conf` - NGINX configuration template

## Usage

1. Copy the template file
2. Remove the `.template` from the filename
3. Customize the values for your specific use case
4. Ensure no sensitive information is included before sharing
